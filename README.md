# AirShop
Stack: React, Redux, TypeScript, Lodash , Webpack, Bootstrap 10, Firebase, Redux-Saga

  [Online demo](https://gifted-feynman-f6f718.netlify.com/)


### Usage

```
git clone https://github.com/Lavitr/Jest-testing-SAGA.git
cd AirShop
npm install
npm start
```

### Available Commands

```
 npm run dev
 npm run build  
```
